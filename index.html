Om:bulari

Om:bulari is a vocab flashcard study app with voice-command capability that limits learning sessions, uses the SRS method of learning, and provides a tailored, interactive and immersive learning experience. The process involved competitive analysis, user research, persona development, user flows, rapid prototyping, and usability testing.

Ink Tank

Ink Tank is a tattoo inspiration and artist finder web app created to minimise tattoo regret. It gives tattoo Enthusiasts a way to search for and preview designs, locate nearby Artists, and feel supported in their search for the perfect artist/design combo so that they don't have "buyer's remorse". The process involved competitive analysis, user research, persona development, user flows, card sorts, rapid prototyping, usability and preference testing, design system development, and UI development.
